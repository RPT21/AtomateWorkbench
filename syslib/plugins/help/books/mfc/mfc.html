Mass Flow Controller

The MFC's link to MKS 647B/C or 146C hardware driver to control gas flow during a run.

Workbench additions

Device Editor

Choose Recipe > Recipe Options and click Add. An "MFC" appears in the type column of the list. Double click this type or select it and click Edit. The device editor window appears.

The User Interface panel contains:

  • Label: the user-defined device name
  • Grid Editor Field: determines what the entered value means
    • Raw Setpoint (radio button): the nitrogen equivalent flow
    • GCF Converted Setpoint (radio button): the real gas flow (need to set the GCF)

The hardware panel contains:

  • Gas Conversion Factor: coefficient needed to obtain the real gas flow of each specific gas
  • Purge Setpoint: gas flow setpoint after recipe ends or is aborted
  • Purge Duration: duration of purge after recipe ends or is aborted
  • Hardware: the hardware to which this device is linked (need to have the driver) MKS 647B/C or 146C
  • Channel: MFC number, which port it is hooked into on MKS controller 1-8 channel for 647 and 1-4 channel for 146
  • Units: units used to read flow value (sccm, slm, etc.)
  • Range: maximum value that can be achieved for each MFC specified on hardware
