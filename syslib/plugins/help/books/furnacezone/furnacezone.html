Heating Zone

The heating zone or furnace links to Yokogawa UP 150 or Eurotherm hardware driver to control temperature in the reaction chamber.

Workbench additions

Device Editor

Choose Recipe > Recipe Options and click Add. A "heating zone" appears in the type column of the list. Double click this type or select it and click Edit. The device editor window appears.

The User Interface panel contains:

  • Label: the user-defined device name

The hardware panel contains:

  • Range: the full range the heat zone is expected to use
  • Purge Setpoint: setpoint after recipe ends or is aborted
  • Purge Duration: duration of purge after recipe ends or is aborted
  • Hardware: the hardware to which this device is linked (need to have the driver)

Status Monitor

Panel with name device name and a temperature read-out appears.

Grid Editor

One column appears corresponding to the heating element.

Extended Editor

Panel with name device name appears below Step Properties. The display shows the setpoint. The ramp option contains:

  • linear ramp (checkbox): if checked, select a ramp to the setpoint
    • previous setpoint (radio button): starts the ramp from the previous step's setpoint
    • last temperature (radio button): starts the ramp at the most recent temperature
