Devices

"Device" refers to the software representation of hardware. The device driver links a real piece of hardware to a device in the recipe. In general, a device is a general representation and can be linked to a number of different hardware drivers. The user must configure each device to reflect the real piece of hardware. To add a device to an existing recipe, choose Recipe > Recipe Options. In the window "Recipe Options", click the Add button. Choose a device from the list in the window "Add Device". The choice will appear in the "Recipe Options" window. Double click on the new device or select it and click the Edit button. All devices will be pre-configured to the corresponding hardware included in each Atomate system.

The window "Device Editor" is used to modify devices. Under User Interface, the user must specify a name which will appear in the column header of the recipe, like channel number or gas type for each MFC, or furnace for the heating element. The subsequent section, different for each device, has the list Hardware, which gives the user options as to which piece of hardware the device will link.

Editing Devices

To change the configuration for a device, choose Recipe > Recipe Options. Double click on the device or select it and click the Edit button. Options such as device name, color corresponding to device and value range can be modified.

General

Device configurations are specific to the recipe for which they were created. Changing these configurations affects only that recipe.

Devices may be added to the recipe without specifying a piece of hardware for them; however, the recipe cannot be executed.
